Grids

The following are sections with a width of x "cells." This adjusts to whatever the screen size is.
Cells are one of the features of foundation and so they do not require any other links.

This is a twelve cell section in a grid-x.

The class for these 6-cell sections is: "large-6 medium-6 cell," where 6 refers to the number of cells and large/medium refers to the size of the screen.
You may adjust how many cells you'd like a section to be on a larger/smaller screen.
